# 计算机系统概述
## 1.1 计算机硬件的基本组成

### 1.1.1 冯诺依曼结构的基本思想
1. 采用"存储程序" 工作方式
2. 由运算器，控制器，存储器，输入输出设备。五个基本部件组成
3. 存储器不仅能存放数据，也能存放指令，形式上数据和指令没有区别，但计算机能区分他们；控制器能自动执行指令；运算器能进行加减乘除四种基本算术运算，并且也能进行逻辑运算；操作人员可以通过输入/输出设备使用计算机。
4. 计算机内部以二进制形式表示指令和数据；每条指令由操作码和地址码两部分组成，操作码指出操作类型，地址码指出操作数的地址；由一串指令组成程序

### 1.1.2 现代计算机的基本组成

包括：中央处理器，存储器，外部设备和各类总线等
- 中央处理器：CPU
  - 数据通路：指指令执行过程中数据所流经的部件，其包括各类运算部件。其中最重要的是算术逻辑部件(ALU),ALU最基本的部件是加法器
  - 控制器：用来对指令进行译码，生成相应的控制信号，以控制数据通路进行特定的操作
- 存储器：分为内存和外存
  - 内存：主存储器和高速缓冲存储器
  - 外存：辅助存储器和海量后背存储器，通常把系统运行时直接和驻村交换信息的存储器称为辅助存储器，简称辅存
- 外部设备：简称外设，也称为`I/O`设备， 其中`I/O`是输入输出
  - 外设通过`I/O`适配器连接到主机上，`I/O`控制器或`I/O`适配器统称为设备控制器
- 总线：传输信息的通路，用于在部件之间传输信息

## 1.2 计算机软件概述

- 系统软件：有效，安全地使用和管理计算机以及为开发和运行应用软件而提供的各种软件，介于计算机应用和应用程序之间，它与具体应用关系不大。系统软件包括操作系统，语言处理系统，数据库管理系统等
- 应用软件：专门为数据处理，科学计算，事务管理，多媒体处理，工程设计以及过程控制等应用所编写的各类程序

## 1.4 程序开发与执行过程

### 冯诺依曼模型机

- 运算器
  - ACC：累加器，用于存放操作数或运算结果
  - MQ：乘商寄存器，在乘除运算时，存放操作数或运算结果
  - X：通用的操作数寄存器
  - ALU：算术逻辑单元，实现算术运算，逻辑运算
- 控制器
  - CU: 控制单元，分析指令
  - IR: 指令寄存器，存放当前执行的指令
  - PC: 程序计数器，存放下一条指令地址，有自动加一的功能
- 主存储器
  - 存储体：数据在存储体按地址存储
  - MAR：存储地址寄存器，MAR位数反应存储单元的个数
  - MDR：存储数据寄		存器，MDR位数=存储字长
    - 存储单元：每个存放一串二进制代码
    - 存储字：存储单元中二进制代码的组合
    - 存储字长：存储单元中二进制代码的位数
    - 存储元：即存储二进制的电子元件，每个存储元可存放1bit
- 输入设备
- 输出设备

### 计算机的工作过程
高级语言 -> 编译装入主存
初始： 指令,数据存入主存，pc指向第一条指令
从主存中取指令放入IR，PC自动加1，CU分析指令，CU指挥其他部件执行指令

## 1.5 计算机系统性能评价

### 1.5.1定义

吞吐率和响应时间是两个基本指标
吞吐率也称为带宽，响应时间也称为执行时间或等待时间

#### CPU的性能指标

- CPU主频：CPU内数字脉冲信号振荡的频率
  - $$时钟频率=\frac{1}{cpu时钟周期}$$
- CPU时钟周期：一凹一凸为一个周期，单位：微秒纳秒
- CPI: 执行一条指令所需的时钟周期
- $$执行一条指令的耗时=CPI * CPU 时钟周期$$
- $$程序总时钟周期数 = \sum_{i=1}^n (CPI_i * C_i)$$
- CPU执行时间 = CPU时钟周期数/主频 = (指令条数 * CPI) / 主频
- $$IPS=\frac{主频}{平均CPI}$$
- FLOPS: 每秒执行多少次浮点运算。
  - 单位：
    - $$K=Kilo=千=10^3$$
    - $$M=Million=百万=10^6$$
    - $$G=Giaga=十亿=10^9$$
    - $$T=Tera=万亿=10^12$$
- 时钟周期数 = cpu执行时间 * 时钟频率

#### 系统整体的性能指标
数据通路带宽: 数据总线一次所能并行传送信息的位数(各硬件部件通过数据总线传输数据)
吞吐量: 系统在单位时间内处理请求的数量
相应时间: 用户向计算机发送一个请i去,到系统对该请求做出响应并获得他所需要的结果的等待时间

基准程序: 用来测量计算机处理速度的一种实用程序,以便于被测量的计算机性能可以与运行相同程序的其他计算机性能进行比较
