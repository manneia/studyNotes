## 三、简答题

### 第1题：冯·诺依曼计算机的基本思想

**参考答案：**
1. **存储程序原理**：程序和数据都以二进制形式统一存储在存储器中
2. **二进制表示**：计算机内部采用二进制来表示指令和数据
3. **顺序执行**：程序自动地、顺序地执行（除非遇到转移指令）
4. **五大部件**：计算机由运算器、控制器、存储器、输入设备、输出设备组成

### 第2题：指令周期与指令/数据区分

**参考答案：**

**指令周期**：CPU取出并执行一条指令所需的全部时间。

**区分方法**：指令和数据在存储器中以相同的二进制形式存放，无法通过内容本身区分。区分的依据是**所处的阶段**：
- **取指阶段**：CPU从存储器中取出的是**指令**，送入指令寄存器IR
- **执行阶段**：根据指令要求，可能需要从存储器中取出**数据**（操作数）

⚠️ **易错提示**：不是按操作码区分，也不是按存储位置区分。

### 第3题：概念辨析

| 概念 | 含义 | 区别 |
|------|------|------|
| **CPI** | 执行一条指令所需的时钟周期数 | 是一个比值，反映指令复杂程度，单个程序属性 |
| **IPS** | 每秒执行的指令条数 | = 主频/CPI，反映执行速度 |
| **吞吐量** | 单位时间内处理的任务数量 | 系统整体处理能力 |
| **响应时间** | 从提交任务到获得结果的时间 | 用户感知的延迟 |

## 四、计算题

### 解答

**已知条件：**
- 主频 f = 2.5 GHz = 2.5 × 10⁹ Hz
- 指令条数 N = 5 × 10⁹ 条
- 平均 CPI = 2

---

**1. CPU执行时间（4分）**

$$CPU执行时间 = \frac{指令条数 \times CPI}{主频} = \frac{5 \times 10^9 \times 2}{2.5 \times 10^9} = \frac{10 \times 10^9}{2.5 \times 10^9} = 4秒$$

**答：CPU执行时间为 4 秒。**

---

**2. IPS（4分）**

$$IPS = \frac{主频}{CPI} = \frac{2.5 \times 10^9}{2} = 1.25 \times 10^9$$

**答：IPS 为 1.25 × 10⁹ 条/秒。**

---

**3. MIPS（4分）**

$$MIPS = \frac{IPS}{10^6} = \frac{1.25 \times 10^9}{10^6} = 1250$$

**答：MIPS 为 1250。**

---

**4. 主频提升后的执行时间（4分）**

$$新CPU执行时间 = \frac{5 \times 10^9 \times 2}{3.0 \times 10^9} = \frac{10 \times 10^9}{3.0 \times 10^9} ≈ 3.33秒$$

**答：主频提升后，CPU执行时间约为 3.33 秒。**

# 核心公式速记卡

```
┌─────────────────────────────────────────────────────────┐
│                    📌 必背公式                           │
├─────────────────────────────────────────────────────────┤
│  时钟周期 = 1 / 主频                                      │
│                                                         │
│  CPU执行时间 = (指令条数 × CPI) / 主频                     │
│             = CPU时钟周期数 / 主频                        │
│                                                         │
│  IPS = 主频 / CPI                                        │
│                                                         │
│  MIPS = IPS / 10⁶                                       │
│                                                         │
│  CPI = 执行一条指令所需的时钟周期数（不是每秒指令数！）       │
└─────────────────────────────────────────────────────────┘
```

# 重点易错点提醒

1. **指令vs数据**：按**所处阶段**区分，不是按操作码！
2. **PC**：存的是**地址**，不是指令内容
3. **MDR vs MAR**：MDR存数据，MAR存地址
4. **CPI**：是"周期数/指令"，不是"指令数/周期"
5. **FLOPS**：系统整体性能指标，不是单个程序属性
