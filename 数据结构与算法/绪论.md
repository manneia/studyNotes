# 第一章 绪论

程序 = 算法 + 数据结构

## 1.1 基本概念和常用术语

数据是描述客观事物的数
数据元素是数据的基本单位，数据元素是最小单位
在数据结构中，数据的最小不可分割单位是数据项；逻辑上相关的，具有物理意义的若干项数据构成一个数据元素；数据元素作为一个完整的对象是构成数据的基本单位。
数据由多个数据元素组成，数据元素由多个数据项组成

数据结构是带有结构的数据元素的集合，结构是数据元素之间的相互关系，即数据的组织形式。
数据结构可以分为逻辑结构和存储结构。

- 逻辑结构主要是指数据元素之间的逻辑关系
  - 集合，线性结构，树结构和图结构
  - 线性结构一对一，树结构一对多，图结构多对多
- 存储结构主要是指数据元素在计算机中的标识及存储方式

- 数据元素及其关系在计算机内的存储方式，称为数据的存储结构，也称为物理结构
  - 顺序存储，逻辑上相邻，物理位置也相邻
  - 链式存储，逻辑上相邻，物理位置不一定相邻。统筹借助指针来表示逻辑关系
  - 索引存储
  - 散列(哈希)存储：根据数据元素的关键字计算该元素的物理存储位置。通常，使用称为散列表的一维数据作为保存元素的结构

抽象数据类型: 类型加上允许对该类型数据进行的一组操作，称为数据类型。若一个类型所表示的值不可再分解，则称为原子类型，若可以分解，即它是由若干个分量按某种结构组成的

## 1.2 算法

算法是一个由若干确定的(无二义性的)，可执行的步骤组成的肯定能够终止的有序步骤集合

算法：
1. 输入
2. 输出
3. 有穷性
4. 确定性
5. 可行性
